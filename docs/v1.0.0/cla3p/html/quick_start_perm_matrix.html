Permutation Matrix

Permutation matrices are used to rearrange a vector or matrix enties.
Permutation matrices are represented by an array of integers.
So an n-sized PermMatrix is a 1D entity and its individual values lie in [0, n-1] (0-based indexing).
CLA3P uses the cla3p::PxMatrix<cla3p::int_t> class to represent such matrices.
For more information about permutation matrices see here.

Creating permutation matrices

You can create permutation matrices like this:

Default creation

CodeOutput (32bit ints)Output (64bit ints)
#include <iostream>
#include "cla3p/perms.hpp"
int main()
{
/*
* An empty permutation matrix
*/
cla3p::PiMatrix P;
std::cout << P.info("P");
/*
* (3x3) permutation matrix (uninitialized values)
*/
cla3p::PiMatrix Q(3);
std::cout << Q.info("Q");
/*
* Allocate space for a (5x5) P (uninitialized values)
*/
P = cla3p::PiMatrix::init(5);
std::cout << P.info("P");
return 0;
}

Create Identity

CodeOutput (32bit ints)Output (64bit ints)
#include <iostream>
#include "cla3p/perms.hpp"
int main()
{
/*
* (5x5) permutation matrix (identity -> Q[i] = i)
*/
cla3p::PiMatrix Q = cla3p::PiMatrix::identity(5);
std::cout << Q.info("Q") << Q;
return 0;
}

Accessing permutation matrix elements

Examples on how to access/fill values in permutation matrices:

CodeOutput
#include <iostream>
#include "cla3p/perms.hpp"
int main()
{
cla3p::PiMatrix P1(3);
cla3p::PiMatrix P2(3);
cla3p::PiMatrix P3(3);
/*
* Set all values in P1 to 0 (not a valid permutation matrix)
* Set all values in P2 to -1 (not a valid permutation matrix)
* Insert values in P3
*/
P1.fill(0);
P2 = -1;
P3(0) = 2;
P3(1) = 0;
P3(2) = 1;
std::cout << "P1:\n" << P1 << "\n";
std::cout << "P2:\n" << P2 << "\n";
std::cout << "P3:\n" << P3;
return 0;
}
