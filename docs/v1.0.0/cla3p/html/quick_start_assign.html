Copying/Moving/Referencing

Copy constructors and copy assignment operators are disabled.
You can explicitly copy and move an object to another.
You can also shallow-copy an object, be aware of content referencing!

Deep Copying

Copy the contents of a vector or matrix to another. Original object is unchanged.

CodeOutput
#include <iostream>
#include "cla3p/dense.hpp"
int main()
{
cla3p::dns::RfVector X(3);
X = 1;
std::cout << X.info("X") << X;
/*
* Copy X to Y using copy()
*
* Copy assignment:
* cla3p::dns::RfVector Y = X;
* is not allowed
*/
cla3p::dns::RfVector Y = X.copy();
std::cout << Y.info("Y") << Y;
return 0;
}

Moving

Moving contents of a vector or matrix to another. Original object is destroyed.

CodeOutput
#include <iostream>
#include "cla3p/dense.hpp"
int main()
{
cla3p::dns::RfVector X(3);
X = 1;
std::cout << X.info("X") << X;
/*
* Move X to Y
*/
cla3p::dns::RfVector Y = X.move();
std::cout << X.info("X") << X;
std::cout << Y.info("Y") << Y;
return 0;
}

Shallow Copying

Create a copy of a vector or matrix to another. Copied object contains a reference to the contents of the original object. Original object is unchanged.

CodeOutput
#include <iostream>
#include "cla3p/dense.hpp"
int main()
{
cla3p::dns::RfVector X(3);
X = 1;
std::cout << X.info("X") << X;
/*
* Reference Copy X to Y (X & Y share contents)
*/
cla3p::dns::RfVector Y = X.rcopy();
std::cout << Y.info("Y") << Y << "\n";
Y(0) = 0;
Y(2) = 0;
std::cout << "X after zero set:\n" << X;
std::cout << "Y after zero set:\n" << Y;
return 0;
}
