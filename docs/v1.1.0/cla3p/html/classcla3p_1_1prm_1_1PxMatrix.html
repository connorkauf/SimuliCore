Detailed Description

template<typename T_Int>
class cla3p::prm::PxMatrix< T_Int >

The permutation matrix object.

Permutation matrices are represented by an array of (unsigned) integers.
Their purpose is to rearrange indices using a predefined mapping.
So an n-sized PermMatrix is a 1D entity and its individual values lie in [0, n-1] (0-based indexing).

Examples
ex03a_permutation_matrix_create.cpp, ex03b_permutation_matrix_fill.cpp, ex03c_permutation_matrix_create_identity.cpp, ex03d_permutation_matrix_create_random.cpp, ex03e_permutation_matrix_permute_vector.cpp, ex03f_permutation_matrix_permute_matrix.cpp, ex03g_permutation_matrix_permute_matrix_symmetric.cpp, and ex03h_permutation_matrix_permute_permutation_matrix.cpp.
